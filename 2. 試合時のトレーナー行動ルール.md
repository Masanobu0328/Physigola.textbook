# 2. 試合時のトレーナー行動ルール

### ― 選手の命を守る、ルールと判断力 ―

---

## はじめに

試合中のトレーナーは、選手の命と安全を守る**最後の場所**です。その活動は「誰よりも冷静に、誰よりも速く」対応することが求められます。この章では、**FIFAや各国協会が定める国際的なガイドライン**をもとに、現場での立ち振る舞い、判断基準、備えるべき知識を整理します【1】。

---

## ベンチ入りとピッチ立ち入りのルール

### ベンチ入り条件（例：FIFA / JFA）

- **公式登録されたスタッフのみ**がベンチ入り可能
- トレーナー、理学療法士、ドクターなども**事前の名簿提出が必要**
- 登録人数は大会やリーグによって異なる（例：最大7〜11人）【2】

### ピッチへの立ち入り

- **必ず主審の許可が必要**（合図があった後に)
- 対象は**身体的負傷者のみ**、それ以外の介入はNG
- 立ち入り中は**選手に声をかける以外の行為は慎重に**【2】

---

## 緊急対応の原則（Emergency Action Plan）

### トレーナーの役割

- **応急処置の第一人者**：緊急に評価 → 適切な対応
- **生命に関わる対応も想定**（例：心停止、頭部外傷など）
- **復帰可否の判断**：過剰な楽観や復帰強要は嚴禁【3】【4】

### 主なガイドラインからの具体例

| 状況 | 対応の一例 |
| --- | --- |
| 捺挟・打擊などの軽傷 | RICE原則、経過観察 |
| 頭部外傷・脳震盪 | SCAT5などのチェックリストに基づき**即退場・医療機関紹介**が基本【5】【6】 |
| 意識消失・心停止 | CPR【心肺蘇生】・AEDの即時対応【3】 |
| 重症疑い（骨折・脱並） | 安静・固定・抽撮／救急搬送判断【3】 |

**EAP**（Emergency Action Plan）は、事故時に迷いなく対応するための事前計画であり、**高校・大学・プロレベルに関係なく必須**とされます【3】【4】。

現場では、脳震盪による意識消失時に選手が不用意に被害者へ接触するケースも見られますが、これは極めて危険です。**頭部外傷においては、選手・スタッフ全員が対応フローを共有しておくことが事故回避に不可欠です**【6】【8】

---

## トレーナーに求められる判断と行動

| 観点 | 実践ポイント |
| --- | --- |
| 観察 | 表情・仕草・出血・意識の有無を瞬時に確認 |
| 判断 | 「グレーゾーンはアウト」で考える（＝安全優先）【3】【6】 |
| 対応 | 性質に応じて即座に処置／医師判断へ |
| 報告 | チームスタッフ・家族・大会側へ適切に報告 |

---

## 最新の現場対応知識（近年の研究より）

- **リウォームアップ（RWU）**： ハーフタイム中の**再ウォームアップは筋温・神経出力を再活性化し、パフォーマンス低下を抑制する**効果があるとされる【7】。
- **疲労管理・再発予防**：
    
    「**GPS・血液データによるコンディション管理、ウォームダウンの徹底**」は、再受傷のリスクを低下させる【8】。
    
- **脳震盪・頭部外傷対応の強化**：
    
    近年、**選手本人や現場の医療スタッフによる復帰許可判断の禁止**が進んでいます【6】。特に**セカンドインパクト症候群の危険性**は、現場トレーナーが正しく理解しておく必要があります【5】【6】【8】。
