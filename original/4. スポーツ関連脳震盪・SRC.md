# ４．スポーツ関連脳震盪・SRC

～「見逃さない、無理をさせない、回復を導く」トレーナーの責任～

はじめに

脳震盪はサッカーを含む接触スポーツで頻発しますが、「その場では元気」に見えることが多く、見落とされやすい外傷です。しかし、脳震盪は命や長期的な脳機能に関わる重大な医学的緊急事態であり、現場トレーナーの判断力と対応力が問われます【1】【2】。

1. 脳震盪とは？

定義

- 軽度外傷性脳損傷（mTBI）の一種
- 頭部への直接・間接的衝撃により、一時的な脳機能障害が生じる
- CT/MRIでは構造的異常がみられないことが多い【1】

発生リスク

| 原因 | 内容 |
| --- | --- |
| ヘディング時の接触 | 頭の衝撃、地面上の打撃 |
| 転倒 | 地面やポールに頭を打つ |
|  |  |

主な症状

- 頭痛、めまい、吐き気
- ぼーっとする、反応遅れ
- 光音過敏、集中力低下
- 記憶障害、情緒不安定
- バランス感覚の喪失、ぼんやり感
- 意識消失（稀）

これらの症状は受傷直後から数時間にわたり変化するため、単発的な確認ではなく継続的観察が必要です【2】。

# 2. 評価と診断

トレーナーの初期評価

- 受傷場面の観察（衝突や転倒の状況確認）
- 声かけ「今、どこで何してた？」
- 視線の安定性、反応速度、動作のぎこちなさを観察
- SCAT6/症状スクリーニング（SCATを持っていない場合でも、チェックリストで代用）
- 症状が一つでもあれば「即時除外」【2】【3】

SCAT6 / SCAT5の使用

| 構成 | 内容 |
| --- | --- |
| 症状チェック | 頭痛、めまいなど（22項目） |
| 認知・記憶評価 | 単語記憶、集中力、逆唱など |
| 神経機能検査 | 指差し、協調性チェック |
| バランス | BESSや片足立ちテスト |
| 反復評価 | 初回→15分後→1時間後の推移を確認 |

SCAT6は受傷後72時間以内の評価に有効で、7日以降では診断精度が低下するとされています【3】。ただし、日本語公式版はSCAT5までのため、現場運用ではSCAT5で統一し、事前ベースラインとの比較運用が推奨されます【2】。

# 3. 現場での役割と対応

| トレーナーの責任 | 内容 |
| --- | --- |
| 疑わしきは除外 | 確認できなければ、とりあえず止めることが優先 |
| 症状の過少申告 | 語られる症状だけでなく、観察力と周囲の情報収集も重要 |
| 指導者・保護者への対応 | ガイドラインや資料を用いて説明・説得 |
| 医療機関との情報連携 | 受傷時刻、症状経過、初期評価の記録を簡潔に文書化 |
| 経過モニタリング | 同日・翌日・翌々日まで最低3回の経過観察 |

症状が時間差で出現する可能性を考慮し、少なくとも48時間以上の観察と記録を続けることが基本です【1】【3】。

# 4. 段階的復帰（Gradual RTP）

→ 各段階は最低24時間をあける。症状再発時は前段階に戻る。

段階的復帰プロトコルは、国際的ガイドライン【3】に準拠して設計されます。特に脳脊髄液漏出症のように、軽運動までは可能でも強度が上がると症状が再燃するケースでは、より慎重な復帰判断と専門医の介入が重要です。また、近年はVOMS（Vestibular/Ocular Motor Screening）などの補助評価も現場導入が進んでいます【4】。

| ステージ | 内容 | トレーナーの役割 | 目的 |
| --- | --- | --- | --- |
| 【ステージ0】 | 完全安静 | 緊急症状の観察、評価結果の把握 | 脳回復の始動 |
| 【ステージ1】 | 軽度有酸素運動（例：ウォーキング） | 反応の変化を観察・記録 | 血流と神経反応のチェック |
| 【ステージ2】 | サッカードリル等（非接触） | 心拍変動・平衡感覚の再評価 | 競技特性への再順応 |
| 【ステージ3】 | 通常トレーニング（非接触） | チーム復帰前の観察と安全管理 | 社会的復帰の評価 |
| 【ステージ4】 | 接触ありのトレーニング（医師承認） | バランス・思考反応の再チェック | 実戦状況への最終調整 |
| 【ステージ5】 | 実戦復帰 | 試合時の心理・身体反応を観察 | 本格復帰と再評価 |

# 5. 再受傷・慢性化リスク

- 脳震盪を短期間で繰り返すと重症化のリスクが大きくなる
- 慢性外傷性脳症（CTE）や記憶障害、感情障害などにつながる【1】【2】
- 一度受傷した選手のフォローアップは1週間以上継続するのが望ましい
- セカンドインパクト症候群は、回復しきっていない状態で再受傷することで重篤な脳腫脹や死に至るリスクを伴う合併症として、国際的にも厳重に警告されています【3】

# 6. 予防・教育もトレーナーの仕事

| 活動 | 内容 |
| --- | --- |
| ルール教育 | 年齢別に応じた安全プレーの普及（例：U12のヘディング制限） |
| 保護具の推奨 | マウスガードやヘッドギアの紹介（特に女性選手） |
| 年齢別リスク指導 | 学生世代ほど神経回復に時間がかかることを説明 |
| チーム教育 | SCATや段階的復帰ガイドラインの配布と活用法指導 |
| 選手説明会 | 「自覚症状を我慢しない」文化づくり |

「教育と啓発」は医学的リスクだけでなく文化的改善も含めた包括的対応が求められます【1】【2】【3】。

まとめ

「今、プレーさせるべきか？」ではなく、「5年後に健康な脳で立てるか」を問うのがトレーナーの未来責任。競技ができなくなるだけでなく、命を落とす可能性がある。
